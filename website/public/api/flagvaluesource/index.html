FlagValueSource

A simple protocol that describes a source of FlagValues

public protocol FlagValueSource

For more information and examples on creating custom FlagValueSources please see the full documentation.

Requirements

name

The name of the source. Used by flag editors like Vexillographer

var name:​ String

flagValue(key:​)

Provide a way to fetch values

func flagValue<Value>(key:​ String) -> Value? where Value:​ FlagValue

setFlagValue(_:​key:​)

And to save values – if your source does not support saving just do nothing

func setFlagValue<Value>(_ value:​ Value?, key:​ String) throws where Value:​ FlagValue

valuesDidChange

If you’re running on a platform that supports Combine you can optionally support real-time flag updates

var valuesDidChange:​ AnyPublisher<Void, Never>?
