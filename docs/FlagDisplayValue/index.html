Protocol Flag​Display​Value 

public protocol FlagDisplayValue

A convenience protocol used by flag editors like Vexillographer.

Use this with your CaseIterable types when you want to customise the value displayed in the UI.

This is only used by types that are edited with a Picker (eg. those that are CaseIterable. It is not used by types that are edited with a TextField or Toggle.
