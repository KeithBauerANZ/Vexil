VexilConfiguration.CodingKeyStrategy

An enumeration describing how keys should be calculated by Flag and FlagGroups.

enum CodingKeyStrategy

Each Flag and FlagGroup can specify its own behaviour. This is the default behaviour to use when they don’t.

Enumeration Cases ¶

`default` ¶

Follow the default behaviour. This is basically a synonym for .kebabcase

case `default`

kebabcase ¶

Converts the property name into a kebab-case string. e.g. myPropertyName becomes my-property-name

case kebabcase

snakecase ¶

Converts the property name into a snake_case string. e.g. myPropertyName becomes my_property_name
