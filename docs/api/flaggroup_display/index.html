FlagGroup.Display

How to display this group in Vexillographer

enum Display

Enumeration Cases ¶

navigation ¶

Displays this group using a NavigationLink. This is the default.

case navigation

In the navigated view the name is the cell’s display name and the navigated view’s title, and the description is displayed at the top of the navigated view.

section ¶

Displays this group using a Section

case section

The name of this FlagGroup is used as the Section’s header, and the description as the Section’s footer.
