Enumeration Vexil​Configuration.​Coding​Key​Strategy 

enum CodingKeyStrategy

An enumeration describing how keys should be calculated by Flag and FlagGroups.

Each Flag and FlagGroup can specify its own behaviour. This is the default behaviour to use when they don't.
