Name Type Description
coding​Path​Strategy Vexil​Configuration.​Coding​Key​Strategy

How to calculate each Flags "key". Defaults to CodingKeyStrategy.default (aka .kebabcase)
prefix String?

An optional prefix to apply to each calculated key,. This is treated as a separate "level" of the tree. So if your prefix is "magic", your flag keys would be magic.abc-flag
separator String

A separator to use by joined() to combine different levels of the flag tree together.
